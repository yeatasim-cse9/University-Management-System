# Academix - University Management System

A comprehensive University Management System designed to streamline academic and administrative processes.

## Features
- **Student Portal**: Course enrollment, results, attendance, and routine.
- **Teacher Portal**: Class scheduling, grading, and student performance tracking.
- **Admin Dashboard**: Manage departments, courses, students, and faculty.
- **Super Admin**: System-wide settings and configuration.

## Setup
1.  Clone the repository.
2.  Import `database/full_setup_2026.sql` into your MySQL database.
3.  Configure `config/settings.php` and `config/database.php`.
4.  Serve via Apache/Nginx.
